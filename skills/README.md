# 📚 Skills EdgeAngel

Ce dossier contient les **22 skills `ea-*`** maintenus par l'équipe EdgeAngel.

Chaque skill est un dossier contenant un fichier `SKILL.md` que l'IA d'Antigravity lit automatiquement quand la tâche le nécessite.

---

## 🚀 Installation rapide

Copiez-collez ce prompt dans Antigravity et l'IA fera le reste :

```
Clone le repo GitHub edgeangel/antigravity-toolkit et copie tous les skills ea-* dans mon dossier skills Antigravity.

Commandes à exécuter :
1. git clone https://github.com/edgeangel/antigravity-toolkit.git /tmp/antigravity-toolkit
2. cp -r /tmp/antigravity-toolkit/skills/ea-* ~/.gemini/antigravity/skills/
3. rm -rf /tmp/antigravity-toolkit
4. Confirme-moi la liste des skills installés.
```

---

## 🔄 Mise à jour

Même principe, copiez-collez ce prompt pour mettre à jour :

```
Mets à jour les skills EdgeAngel depuis GitHub :
1. git clone https://github.com/edgeangel/antigravity-toolkit.git /tmp/antigravity-toolkit
2. cp -r /tmp/antigravity-toolkit/skills/ea-* ~/.gemini/antigravity/skills/
3. rm -rf /tmp/antigravity-toolkit
4. Confirme-moi que les skills sont à jour.
```

---

## ⚠️ Règles

- **Ne modifiez pas** les skills `ea-*` directement — vos modifications seront écrasées à la prochaine mise à jour
- **Créez vos propres skills** avec un préfixe personnel (ex: `leo-`, `cam-`)
- **Signalez vos feedbacks** via les [Issues GitHub](https://github.com/edgeangel/antigravity-toolkit/issues)

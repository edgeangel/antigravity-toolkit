# Frameworks Créatifs — Référence

> Frameworks détaillés pour la génération d'angles de communication. Chargé par `ea-studio-creative-strategist`.

## PAS — Problem, Agitate, Solve

**Registre** : Rationnel / Direct Response
**Quand l'utiliser** : Quand la cible a un problème identifié et que le produit apporte une solution concrète.

### Structure
1. **Problem** — Nommer le problème. Être spécifique. "Tu perds 3h par semaine à…" pas "Tu as un problème de productivité"
2. **Agitate** — Amplifier la douleur. Montrer les conséquences de ne pas agir. Créer de l'inconfort
3. **Solve** — Présenter la solution. Montrer le chemin vers le soulagement

### Exemple
> **P** : "Tu dépenses 5 000€/mois en ads sans savoir ce qui marche vraiment."
> **A** : "Pendant ce temps, ton concurrent sait exactement quel euro lui rapporte. Chaque jour qui passe, tu brûles du budget."
> **S** : "Avec [Produit], chaque euro est tracé. Tu sais. Tu décides."

## Storytelling Before-After

**Registre** : Émotionnel / Transformation
**Quand l'utiliser** : Quand le produit transforme la vie/le business du client et qu'on peut montrer le delta.

### Structure
1. **Before** — La situation actuelle (frustration, manque, galère)
2. **Transformation** — Le moment charnière (découverte du produit/service)
3. **After** — La nouvelle réalité (bénéfices concrets, émotions positives)

### Variantes
- **Hero's Journey** : Le client est le héros, la marque est le guide (à la StoryBrand)
- **Day-in-the-life** : Montrer une journée type avant/après
- **Témoignage narratif** : "Il y a 6 mois, j'étais…"

## Social Proof + FOMO

**Registre** : Validation sociale / Urgence
**Quand l'utiliser** : Quand la marque a des preuves (clients, chiffres, témoignages) et que la cible a besoin de réassurance.

### Mécaniques
- **Chiffres** : "12 000 clients font confiance à…", "+340% de conversions"
- **Témoignages** : Citation client + résultat concret
- **Autorité** : Certifications, partenariats, mentions presse
- **FOMO temporel** : "Les 50 premiers", "Offre limitée", "Dernières places"
- **FOMO social** : "Vos concurrents utilisent déjà…"

### Règle
Le Social Proof doit être **vérifiable**. Pas de chiffres inventés. Si la marque n'a pas de données dures, utiliser des signaux indirects (ancienneté, nombre de projets, satisfaction clients).

## Pattern Interrupt

**Registre** : Provocation / Rupture
**Quand l'utiliser** : Quand le marché est saturé de messages similaires. Quand il faut casser le scroll. Quand la marque peut se permettre un ton audacieux.

### Mécaniques
- **Contre-intuition** : "Arrêtez de [ce que tout le monde fait]"
- **Question dérangeante** : "Et si [croyance commune] était faux ?"
- **Statistique choc** : "72% des [cibles] font [erreur]"
- **Paradoxe** : "Plus vous [action], moins vous [résultat attendu]"
- **Confession** : "On va être honnêtes : [vérité inconfortable]"

### Garde-fous
- La provocation doit être **justifiée** par la suite du message
- Jamais de provocation gratuite sans substance derrière
- Le ton doit rester aligné avec la marque — certaines marques ne peuvent pas provoquer

## Value-First / Lead Magnet

**Registre** : Utilitaire / Éducatif
**Quand l'utiliser** : Quand la cible est en phase de considération, quand le produit est complexe, quand on veut générer des leads qualifiés.

### Mécaniques
- **Checklist** : "Les 5 erreurs à éviter quand…"
- **Template** : "Le template que nos clients utilisent pour…"
- **Mini-audit** : "Testez votre [domain] en 30 secondes"
- **Tuto rapide** : "Comment [résultat] en 3 étapes"
- **Data originale** : "Nous avons analysé 1 000 [sujets], voici ce qu'on a trouvé"

### Règle
La valeur donnée doit être **réellement utile** — pas un teaser vide. Le lecteur doit repartir avec un quick win même s'il n'achète pas.

## AIDA (variante pour les ads)

**Registre** : Classique / Polyvalent
**Quand l'utiliser** : En complément quand les 5 frameworks principaux ne suffisent pas, ou pour structurer un angle qui ne rentre pas dans une catégorie.

### Structure
1. **Attention** — Le hook (première ligne / premier visuel)
2. **Interest** — Développer le sujet (pourquoi le lecteur devrait s'en soucier)
3. **Desire** — Créer l'envie (montrer le bénéfice transformationnel)
4. **Action** — Le CTA (clair, précis, urgent)

## Règles Transversales

| Règle | Application |
|---|---|
| **Headline ≤ 8 mots** | Chaque angle doit pouvoir être résumé en 8 mots max |
| **1 idée = 1 angle** | Ne pas mélanger 2 mécaniques dans un seul angle |
| **Mobile-first** | Tout est pensé pour être lu/vu sur mobile en 3 secondes |
| **Test de remplacement** | Si le nom de marque peut être remplacé par un concurrent, l'angle est trop générique |
| **CTA spécifique** | "Télécharger le guide" > "En savoir plus" > "Cliquez ici" |
